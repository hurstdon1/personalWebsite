Keyword Generator

The Keyword Generator Program was something I created as part of a group project for a software engineering course. The project uses a series of REST APIs to start with the name of a video game before ultimately delivering a word/image cloud of words associated with the game.

My portion of the project does the following:

  • Takes user input in the form of a text document
  • This input can be sent via an api route or uploaded directly to the site via a dropzone
  • The program then breaks apart the json data and extracts a list of keywords
  • The program checks if each move will put the user in check or has put the opponent in checkmate

Check out the program on my github!

XiangQi Game

The Xiangqi Game program is an implementation of an abstract board game.

It has the following functionality

  • Initializes a game board and fills it with unique pieces
  • Allows the user to make moves by calling a function and passing in the starting/ending locations
  • The program checks that the move is valid and if it is, implements the move
  • The program checks if each move will put the user in check or has put the opponent in checkmate
